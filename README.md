# fotmob-images

Team and league logos from FotMob.

## Structure

```
team/    - Team logos (2935 files, 1-30000 range)
league/  - League logos (605 files, 1-20000 range)
```

## Usage

```
https://raw.githubusercontent.com/yosintv2/fmobimg/main/team/{id}.png
https://raw.githubusercontent.com/yosintv2/fmobimg/main/league/{id}.png
```

## Update

Run `python3 download.py` to scan for new images up to ID 30000.
